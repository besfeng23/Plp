# PLP Phase 1 Visual Proof Plan

Phase 1 is the visual credibility phase.

The current code removes public grey placeholder blocks and replaces them with a controlled PLP aerial-image treatment. That improves the live MVP immediately, but the final Aman / Raffles / Four Seasons standard still requires a complete real image library.

## Image Standard

All public images should be:

- real PLP or real Boracay-location photography
- high resolution
- naturally lit
- color graded consistently
- quiet, private, and premium
- exported as WebP or AVIF
- supplied with meaningful alt text

Avoid:

- generic stock hotel photos
- low-resolution screenshots
- visible grey blocks
- images with watermarks
- exaggerated social-media filters

## Homepage Images Needed

- cinematic hillside villa hero
- golden-hour or dusk villa image
- resort atmosphere image
- pool or terrace detail
- island experience image
- private dining image

## Phase 1 Acceptance Criteria

Phase 1 is fully complete only when:

- no public page contains a grey placeholder
- each room has real imagery
- each experience category has real imagery
- homepage hero uses real visual media
- mobile image crops are checked
- meaningful alt text exists
- image sizes are optimized

Until the full photo set is available, the current visual fallback is Phase 1A, not full Phase 1B.
